# PRE Examen Practico: SafeNet ASI

**Nombre:** Lozada Ramirez Angel Israel  
**Materia:** APTI 

## 🛡️ Nombre del Proyecto
**SafeNet ASI: Portal de Cultura Cibersegura**

## 🎯 Objetivo
Desarrollar una plataforma digital centralizada para la gestión de consejos, tips y legislación sobre seguridad informática personal, facilitando el acceso a la información y el registro de nuevas medidas preventivas.

## 🚀 Alcance
- Implementación de un formulario web para el alta de consejos de seguridad.
- Creación de una base de datos MySQL para el almacenamiento persistente de la información.
- Visualización dinámica de los datos registrados (Consultas).
- Uso de arquitectura Java Web (JSP, Servlets, DAO, JDBC).

## 🛠️ Prompts utilizados (Evidencia GEMINI)
1. "Ayúdame a diseñar el objetivo y alcance de un proyecto de seguridad informática personal con 5 campos."
2. "Genera el código para la conexión, modelo y controlador en Java Web."
3. "Diseña una interfaz index.jsp usando Bootstrap para altas y consultas."
